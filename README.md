# DevOps and Cloud Portfolio

Welcome to my DevOps and Cloud Portfolio! This static website showcases my experience, projects, and GitHub repositories in the fields of DevOps and cloud technologies.

## Project Overview

This portfolio is designed to highlight my skills and projects, providing a comprehensive view of my work in the DevOps and cloud domains. The website includes the following sections:

- **Home**: The landing page that introduces visitors to my portfolio.
- **About**: A detailed overview of my experience and expertise in DevOps and cloud technologies.
- **Projects**: A showcase of my projects, dynamically loaded from a JSON file.
- **Contact**: A section for visitors to reach out to me.

## Technologies Used

- HTML, CSS, and JavaScript for the front-end development.
- JSON for data management.
- GitHub Pages for hosting the website.
- GitHub Actions for CI/CD deployment.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/devops-cloud-portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd devopsbysantosh
   ```
3. run  `npm run build` to build the code locally

4. Once packages are built, run this python command  `python3 -m http.server 8000 --directory docs` locally and access the localhost 8000 to view the website 

## Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions. Any changes pushed to the main branch will trigger the deployment workflow defined in `.github/workflows/deploy.yml`.
